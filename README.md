# Landing Page - AI Interview Trainer

Projeto estatico separado para divulgar o aplicativo.

## Como testar localmente

Abra `index.html` no navegador.

## O que trocar antes de publicar

- Substitua `https://seu-app.vercel.app` pela URL real do aplicativo.
- Substitua o bloco visual de demo por um embed do YouTube, Loom ou Vimeo.
- Ajuste textos, beneficios e CTA conforme seu publico.

## Ideia usada como referencia

A landing segue a logica de MVP do artigo: explicar a proposta, mostrar uma demo, coletar interesse e levar usuarios para testar o produto antes de investir em recursos demais.
